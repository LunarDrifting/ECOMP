## Accomplishments
- Added tenant-scoped approval lookup helper in db wrapper for task-level approval checks.
- Updated `markTaskDone` approval enforcement:
  - `approvalPolicy === NONE` passes approval check
  - non-`NONE` policies (`SINGLE`, `SEQUENTIAL`, `PARALLEL`, `QUORUM`) use same minimum rule: at least one `APPROVED` approval row
  - if approval fails, throws deterministic error and does not mutate task state or call resolver
  - successful path proceeds with DONE update and resolver handoff
  - response now includes `approvalCheckPassed`
- Updated completion route to map approval enforcement errors to 409 conflict while staying thin and Prisma-free.
