- Added tenant-scoped db wrapper support for:
  - listing `TemplateDependencyDefinition` by `templateVersionId + tenantId`
  - listing existing runtime dependencies by task ids
  - creating runtime dependency rows with tenant task-scope validation
- Upgraded `instantiateTemplateForEco` to:
  - load blueprint dependency definitions
